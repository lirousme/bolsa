/api/quote/{tickers} — Guia Completo

Endpoint principal para cotação, histórico, dividendos, fundamentos e módulos financeiros detalhados.

URL base: https://brapi.dev/api/quote/{tickers}

Ações de teste sem token: PETR4, MGLU3, VALE3, ITUB4.

Parâmetros (7 no total)

1) tickers (path)

Define um ou mais ativos separados por vírgula.

2) token (query/header)

Autenticação para tickers fora da lista de teste.

3) range

Período do histórico de preço.

4) interval

Granularidade temporal do histórico.

5) fundamental

Inclui dados P/L e LPA.

6) dividends

Inclui histórico de dividendos e JCP.

7) modules

Acopla módulos financeiros avançados.
