Parâmetro: token (query/header)

Tipo: string opcional na query, obrigatório para ativos fora da lista de teste.

Também aceito via header: Authorization: Bearer SEU_TOKEN

Regra: se misturar ticker de teste com ticker comum, token passa a ser obrigatório para toda requisição.
